# ⚡ Quick Start - Deploy lên GitHub Pages

## 🚀 Đơn giản nhất: Chỉ cần 4 lệnh!

```bash
# 1. Thêm remote GitHub (chỉ chạy lần đầu)
git remote add origin https://github.com/iamprmgvyt/maya-bot.git

# 2. Add tất cả file
git add .

# 3. Commit
git commit -m "Initial commit: Maya Bot website"

# 4. Push lên GitHub
git branch -M main
git push -u origin main
```

## ⚙️ Kích hoạt GitHub Pages (Chỉ làm 1 lần)

1. Mở: https://github.com/iamprmgvyt/maya-bot/settings/pages
2. **Source**: Chọn **GitHub Actions**
3. Click **Save**

## 🌐 Website URL sau khi deploy:

```
https://iamprmgvyt.github.io/maya-bot/
```

## 📝 Cập nhật sau này:

```bash
git add .
git commit -m "Mô tả thay đổi"
git push origin main
```

GitHub sẽ tự động deploy! ✨
